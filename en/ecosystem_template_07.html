Description
Airgate develops Rodi-X software applications designed for direct installation onto Hanwha Robotics systems. These applications enhance the robots' functionalities, enabling end-customers to effortlessly integrate and utilize various ecosystems with their Hanwha Robotics equipment.

Software Development Process

1. The initial telco meeting aims to outline the high-level scope of the Rodi-X software development project.
2. The customer collaborates with Airgate to create a product specification, either based on use cases or through joint effort.
3. Following the product specification, Airgate sends an initial offer to the customer for preliminary approval.
4. A kick-off telco meeting with the customer is scheduled to delve into the product specification details and establish the development process.
5. Upon acceptance of the final offer by the customer, Airgate receives the hardware documentation and/or hardware itself for mounting onto the Airgate robot for testing during the development period.
6. Delivery includes the Rodi-X software application, along with documentation and source code.
7. A standard 3-month warranty is provided for a bug-free, stable Rodi-X software application.
